# AIFORUS CLT+ — GitHub Pages 배포 가이드

## 파일 구조
```
/
├── index.html       ← 대시보드
├── chances.html     ← 지원사업 (필터 + 키워드)
├── news.html        ← 지역·업종 뉴스
└── css/
    └── common.css
```

## GitHub Pages 배포 방법
1. GitHub에 새 레포 생성 (예: `aiforus-clt-demo`)
2. 이 폴더의 파일 전체 push
3. Settings → Pages → Branch: main / root → Save
4. `https://{username}.github.io/aiforus-clt-demo/` 에서 접근

## 인터랙션
- 사이드바 메뉴로 페이지 이동
- 필터 칩 클릭 선택
- 우측 플로팅 버튼: 마이페이지 / 키워드관리 패널 슬라이드
- 알림 토글 on/off
